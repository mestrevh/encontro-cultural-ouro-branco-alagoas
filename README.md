# Encontro Cultural OB AL (Ouro Branco - Alagoas)

Esse é um evento que busca unir culturas como Hip Hop, Free Step e entre outras no sertão alagoano.

## Desafios

1. Criar o site utilizando apenas a tecnologia `Gemini CLI`
2. O site deve ser multiplataforma (responsivo)
3. Deve conter a história do evento, informações do evento atual e o cronograma

## Automação

Buscar sempre utilizar os agentes para automatizar o código para análise no futuro

## FRONT-END

* Execução em ambiente de desenvolvimento:

1. Execute o comando abaixo
   
```
bash
cd frontend
npm install
npm run dev
```
